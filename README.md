# Network Traffic Analyzer (Python)

This project analyzes basic network traffic data using Python.
It is designed as an introductory cybersecurity and computer science project.

## Skills Used
- Python
- Data analysis
- Basic networking concepts
- Security awareness

## Project Goals
- Read network traffic data
- Analyze source and destination IPs
- Identify unusual traffic patterns

## Background
This project combines economics, data analysis, and computer science
with a focus on cybersecurity.
